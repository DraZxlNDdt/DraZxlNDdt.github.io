翻翻翻1

禁毒这件事基本上是我们全部人民所赞成的，毒品这也是我们民族的一块伤疤。

那为什么国家不禁烟呢？
我们要想一下他们之间的差别在哪。
抽烟和吸毒差别主要在危害性(谁死得快一点).

那么我们先想一件事情：以前的计划生育和现在的二胎政策的目的是什么？
应该不是为了控制人口，而是为了改变人口结构。

高中生物教会了我们：一个种群的年龄结构取决于迁入迁出，出生死亡率。我们人对动物种群的年龄结构的控制主要就是改变它们的出生率和死亡率。

由计划生育政策改为二胎政策是为了提高生育率。要想干预现在的人口结构自然还有另外一条路——增大年龄大的人的死亡率。那么问题看起来就比较有趣了：抽烟可以让你在退休前保有劳动力，而吸毒不能。

其实计划死亡是很早之前就提出的一个概念了…

计划死亡比计划生育好，可以解决老龄化，又能控制人口规模，不会导致资源不够用，一举两得。而计划生育只能解决老龄化，不能控制人口规模，而且收效要在很多年后。
计划死亡可以隐形施行：降低医疗水平，提升医疗费用，鼓励烟酒行业。既不会影响社会稳定，又能增加烟酒税，减少医疗支出。——转自知乎问题 https://www.zhihu.com/question/52110344

发表于 2019-11-17 更新于 2019-12-07 分类于 日志

今天早上九点20起床，真的颓废！

起床后就去icenter了，线掉了，灰度传感器坏了一个，用了一上午才弄好。下午的时候趴在地上写出了让小车识别转角的程序。感觉还不错。然而加了机械臂以后出现了几个问题

  1. 打滑
  2. 由于重量的增加，速度有点问题
  3. 由于加了一层，灰度传感器可能出锅（光线问题？）
  4. 由于比赛场地的黑线比较宽，所以用两个相邻的灰度传感器实际上并没有什么用。
  5. 比赛要求最后小车回到起点，所以现在需要在小车的四条边都装传感器。

解决方案

  1. 两个车的轮子都要加弹簧！
  2. 第二层的板子开缝
  3. 底盘重新设计

晚上又去吃了清青披萨，sigh~怎么这么快乐啊？回到寝室，ddl之前水完了人计导的作业，现在爆肝线代，加油，奥利给！
